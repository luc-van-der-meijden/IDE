Visual Studio

Microsoft Visual Studio est une suite de logiciels de développement
pour Windows et mac OS conçue par Microsoft.

La dernière version s'appelle Visual Studio 2019.

Visual Studio est un ensemble complet d'outils de développement permettant
de générer des applications web ASP.NET, des services web XML,
des applications bureautiques et des applications mobiles.

Adobe Dreamweaver

Dreamweaver est un éditeur de site web WYSIWYG pour Microsoft Windows,
et Mac OS X créé en 1997, commercialisé par Macromedia puis
Adobe Systems sous licence utilisateur final.

Dreamweaver fut l'un des premiers éditeurs HTML de type « tel affichage,
tel résultat », mais également l'un des premiers à intégrer
un gestionnaire de site (CyberStudio GoLive étant le premier).

Xcode

Xcode est un environnement de développement pour macOS,
ainsi que pour iOS, watchOS et tvOS.

L'API Cocoa permet de programmer avec les langages suivants :
- Objective-C
- Ruby
- Swift (nouveau langage de programmation d'Apple présenté à la WWDC 2014).
