Xcode 11

De meilleures applications. Moins de code.

Des outils que vous adorerez utiliser.

L’IDE Xcode est au centre de l’expérience de développement d’Apple. Bien intégré aux frameworks Cocoa et Cocoa Touch,
Xcode est un environnement incroyablement productif pour la création d'applications pour Mac, iPhone, iPad, Apple Watch et Apple TV .

Xcode 11

Xcode 11 inclut tout ce dont vous avez besoin pour créer de superbes applications
et pour amener vos applications sur encore plus de périphériques.

Tirez parti de SwiftUI, une toute nouvelle infrastructure d'interface
utilisateur dotée d'une syntaxe déclarative de Swift.

Apportez votre application iPad sur Mac en un seul clic.
Et avec la prise en charge des packages Swift,
Xcode 11 vous permet de partager du code entre toutes vos applications
ou d'utiliser des packages créés par la communauté.

SwiftUI

SwiftUI est un moyen innovant et extrêmement simple de créer des interfaces utilisateur sur toutes les plateformes Apple avec la puissance de Swift. Créez des interfaces utilisateur pour tous les appareils Apple à l'aide d'un seul ensemble d'outils et d'API. Avec une syntaxe Swift déclarative facile à lire et à écrire, SwiftUI fonctionne de manière transparente avec les nouveaux outils de conception Xcode pour que votre code et votre conception soient parfaitement synchronisés. SwiftUI est véritablement natif. Vos applications accèdent donc directement aux technologies éprouvées de chaque plate-forme pour mettre en œuvre de manière magnifique tout ce que les utilisateurs aiment de l'écosystème Apple.

Apportez votre application iPad sur Mac

Avec Xcode, il est facile de prendre une longueur d’avance sur la transformation de votre application iPad existante en application Mac native. Vos applications Mac et iPad partagent le même projet et le même code source. Par conséquent, toute modification apportée est répercutée sur les deux plates-formes. Et votre application Mac nouvellement créée fonctionne en mode natif, en utilisant les mêmes infrastructures, ressources et même environnement d'exécution que les applications conçues spécialement pour Mac.

Forfaits rapides et rapides

Swift 5 est désormais intégré à toutes les plates-formes Apple et l'interface binaire pour Swift est stable. Vos applications seront plus petites, seront téléchargées plus rapidement et continueront à fonctionner à mesure que Swift continue d'évoluer.

Les packages Swift sont intégrés dans l'ensemble de Xcode 11, ce qui simplifie incroyablement l'utilisation d'un package dans vos applications pour les plates-formes Apple. Ajoutez simplement une nouvelle dépendance de paquet pour ajouter un paquet externe à votre projet., Puis clonez le paquet depuis GitHub, Bitbucket, GitLab ou votre propre hôte de code. Xcode vérifie les dépendances des packages et affiche tous les packages que vous utilisez directement dans le navigateur.
